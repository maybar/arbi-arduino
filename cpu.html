El rover ARBI1 usa una tarjeta controladora SainSamart UNO R3 ATmega328-AU compatible con Arduino UNO. El microcontrolador Atmel de la tarjeta ejecuta el programa a 20MHz, el cual se implmenta toda las funcionalidades de movimiento, diagnostico, control y comunicaciones del rover.

Especificaciones

  • ATmega328-AU microcontroller
  • Input voltage - 7-12V
  • 5V Electric current : 500MA
  • 3.3V Electric current : 50MA
  • 14 Digital I/O Pins (6 PWM outputs)
  • 8 Analog Inputs
  • 32k Flash Memory
  • 16Mhz Clock Speed
